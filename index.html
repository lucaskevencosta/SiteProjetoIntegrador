PROJETO INTEGRADOR

Projeto Integrador é um projeto multidisciplinar desenvolvido pelos alunos do curso Gestão da Tecnologia da Informação da Faculdade de Tecnologia Senac Goiás ao final de cada período letivo.

PORTIFÓLIO DIGITAL

Com os produtos obitidos a cada módulo do curso, cada aluno forma o seu Portifólio Digital. o Portifólio Digital é o conjunto dos trabalhos desenvolvidos durante os projetos integradores de cada período.

O Site

Para tornar este portifólio público, a faculdade disponibiliza um espaço para cada aluno desenvolver um web site e hospedar todos os trabalhos que farão parte do seu Portifólio Digital.
